# Team Wisdom

Reusable patterns and heuristics learned through work. NOT transcripts — each entry is a distilled, actionable insight.

## Patterns

**Pattern: Feature Flag Staging** **Context:** Browser-only products with tight iteration needs. Decouple feature merges from release cycles by shipping features disabled. Enables small PRs, parallel release work (versioning/docs), and QA-time feature validation without deploy overhead. Applies when backlog flow matters more than synchronized releases.
